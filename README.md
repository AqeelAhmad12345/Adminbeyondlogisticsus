# Beyond Logistics

Static marketing site for Beyond Logistics (beyondlogisticsus / cartalone.com), built per the
client's Website Rebrand brief. Pure HTML/CSS/JS, no build step required — deploys via GitHub
Pages using the existing `CNAME`.

## Structure

- `index.html` — Home
- `about.html` — About / Our Vision / Values / CEO message
- `services.html` — Services overview
- `services/` — Drive-Away, FTL & LTL, Intermodal, Expedited, Refrigerated, Flatbed
- `drivers.html`, `carriers.html` — Network recruitment + application/forms/packet placeholders
- `load-board.html` — Load board placeholder (pending client integration details)
- `contact.html` — Contact + quote request form
- `assets/css/style.css` — Brand design system (colors, type, components)
- `assets/js/main.js` — Sticky nav, mobile menu, dropdowns
- `assets/img/logo.svg`, `favicon.svg` — Recreated Beyond Logistics mark
